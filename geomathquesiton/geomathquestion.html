数学选择题（1–5）

题目 1：一个角度是 90°，它是什么角？

计算方法： 直角是恰好 90° 的角。

笔记： 所有直角三角形都包含一个 90° 的角。

题目 2：如果一条直线的斜率是 0，它表示什么类型的直线？

计算方法： 斜率 m=0 表示 y 不随 x 变化，即水平线。

题目 3：二次函数 y = x^2 的顶点坐标是多少？

计算方法： y = x^2 的顶点为 (0,0)。

题目 4：直角三角形的三个内角之和是多少？

计算方法： 所有三角形内角和均为 180°。

题目 5：若 2x + 3 = 11，则 x 等于多少？

计算方法： 2x = 8, x = 4。

Soalan Pilihan (1–5)

Soalan 1: Sudut 90° ialah jenis sudut apa?

Kaedah: Sudut tegak ialah 90°.

Soalan 2: Jika kecerunan ialah 0, apakah jenis garis itu?

Kaedah: m=0 bermaksud garis mendatar.

Soalan 3: Puncak y = x^2 ialah?

Kaedah: (0,0).

Soalan 4: Jumlah sudut dalaman segitiga ialah?

Kaedah: Semua segitiga berjumlah 180°.

Soalan 5: Jika 2x + 3 = 11, x = ?

Kaedah: x = 4.

Math Multiple Choice (1–5)

Question 1: What type of angle is 90°?

Calculation: Right angle is exactly 90°.

Question 2: If a line has slope 0, what type of line is it?

Calculation: m=0 means horizontal line.

Question 3: What is the vertex of y = x^2?

Calculation: Vertex at (0,0).

Question 4: What is the sum of interior angles of a triangle?

Calculation: Sum is 180°.

Question 5: If 2x + 3 = 11, what is x?

Calculation: x = 4.
